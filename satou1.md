# オブジェクト指向問題の作成01

1. 以下のフィールドを持つFlowerクラスを作成してください。

```
color		:花の色
seedNum	:種の数
water		:水の量
bloom		:咲いた花の数
```

2. 実行例を実現できるようにコンストラクタとメソッドを追記してください。

	汲む水の量は`3`で、最大値が`3`を超えないようにしてください。  
また、咲いた花は`*`で、種は`.`で表してください。

3. 実行例のように出力される、mainメソッドを持つFlowerAppクラスを作成してください
